# my-dotfiles

A personal repo for keeping Tanapol's dotfiles

## Procedure
1. `cd ~`
1. `git clone git@gitlab.com:tanapol/my-dotfiles.git dotfiles`
1. `cd dotfiles` 
1. `./install-dry.sh` to check the symlink creation to your $HOME directory
1. `./install.sh` to install
